Wallet電子錢包圖片限制

Apple Wallet 的 圖片資源 有嚴格的尺寸和格式限制，以確保顯示效果和效能，以下是各圖片的詳細規範：

icon.png（必須）

  • 用途：主要標誌圖示。
  • 尺寸：180x180 像素。
  • 格式：PNG。
    備註：這是必需的圖片，並會顯示在通知和鎖定畫面等位置。

icon.png（必須）

  • 用途：主要標誌圖示。
  • 尺寸：180x180 像素。
  • 格式：PNG。
    備註：這是必需的圖片，並會顯示在通知和鎖定畫面等位置。

background.png

  • 用途：整個 Pass 的背景圖片。
  • 尺寸：最大 180x220 像素（推薦），具體高度可根據需要調整。
  • 格式：PNG。
    備註：會被前景文字蓋住，避免使用複雜圖案。

strip.png

  • 用途：顯示在 Pass 頂部的圖示條帶。
  • 尺寸：624x246 像素。
  • 格式：PNG。
    備註：主要用於標識 Pass 的品牌，適合設置為寬條橫幅圖。

thumbnail.png

  • 用途：顯示在 Pass 主要內容區域的小縮略圖。
  • 尺寸：90x90 像素（推薦）。
  • 格式：PNG。
    備註：用於提供額外的視覺提示，例如活動地點或公司標誌。

footer.png

  • 用途：顯示在 Pass 底部的圖示。
  • 尺寸：286x15 像素。
  • 格式：PNG。
    備註：用於顯示小型標語或附加資訊。

小結

圖片名稱 尺寸 (寬x高) 必需 格式
icon.png 180x180 是 PNG
logo.png 160x50 否 PNG
background.png 180x220（推薦） 否 PNG
strip.png 624x246 否 PNG
thumbnail.png 90x90 否 PNG
footer.png 286x15 否 PNG

注意事項：

  • 格式：所有圖片必須為 PNG 格式，並且支援透明背景。
  • 檔案大小：圖片應盡量小，避免超過 1MB，以提升 Pass 的效能。
  • 顏色模式：圖片應使用 RGB 顏色模式，不支援 CMYK。
