xcode 的檔案旁邊有M是什麼意思？

在 Xcode 中，檔案旁邊的 “M” 通常表示該檔案已被修改(Modified)。

Xcode 使用 Git 作為其內置的版本控制系統，可以追蹤並控制你的程式碼變化。這些變化會在 Xcode 的界面中用不同的標記來表示：

  • “M” 表示該檔案已被修改（Modified）。這意味著你對該檔案作了一些變更，但這些變更還沒有被提交到你的版本控制系統中。
  • “A” 表示該檔案已被加入（Added）到版本控制系統中，等待被提交。
  • “?” 表示該檔案未被版本控制系統追蹤。
  • “D” 表示該檔案已被刪除（Deleted）但變更還未被提交。

如果你看到一個檔案旁邊有 “M” 的標記，這意味著你可能需要提交這個檔案的變更，以便將其納入版本控制。
